Bank Management System (C++)
 Overview

The Bank Management System is a console-based application developed in C++ that simulates basic banking operations. It allows users to create accounts, manage transactions, and maintain customer records efficiently using file handling.

 Features
Create a new bank account
View account details
Deposit money
Withdraw money
Delete account
Update account information
Display all accounts
Secure data storage using files
🛠️ Technologies Used
C++
File Handling (fstream)
Object-Oriented Programming (OOP)
 Project Structure
Bank-Management-System/
│── main.cpp
│── data.txt (or accounts.txt)
│── README.md
⚙️ How It Works
User runs the program
Menu is displayed with multiple banking options
User selects an option
Program performs the operation using file storage
Data is saved permanently in files
